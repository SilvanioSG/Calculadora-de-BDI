Atenção: A CPRB não está incluída no Simples Nacional. Se a obra for desonerada, lembre-se de adicionar a CPRB de 4,5% ao custo ou ao preço.
Dica: Em muitos municípios, o ISS incide apenas sobre a mão de obra. Considere deduzir o valor dos materiais da base de cálculo para um orçamento mais preciso.

Fórmula (Acórdão 2622/2013 – TCU):

BDI = { (1 + AC + SG + R) × (1 + DF) × (1 + L) / (1 - I) } - 1

Onde: AC=Adm. Central, SG=Seguros, R=Riscos, DF=Desp.Financeiras, L=Lucro, I=ISS + PIS/COFINS + CPRB.

* O lucro é antes dos tributos sobre o lucro (IRPJ/CSLL).
